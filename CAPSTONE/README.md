# Temperature Converter — Simple MVC PySide6 App

## Description

A simple desktop application that converts temperatures from **Celsius to Fahrenheit**.  
It demonstrates the **Model–View–Controller (MVC)** architectural pattern using Python and PySide6.

---

## MVC Overview

| Layer | File | Responsibility |
|---|---|---|
| **Model** | `model.py` | Stores the Celsius value; contains the conversion formula. Has **no** PySide6 dependency. |
| **View** | `view.ui` / `view.py` | UI layout created in Qt Designer and compiled with `pyside6-uic`. Contains only widgets — no business logic. |
| **Controller** | `controller.py` | Loads the View, listens to button-click signals, calls the Model, and updates the View with the result. |

```
main.py          ← Entry point (creates QApplication + Controller)
model.py         ← Business logic (pure Python, no Qt)
controller.py    ← Signal–slot connections; bridges View ↔ Model
view.ui          ← Qt Designer source file
view.py          ← Auto-generated by pyside6-uic from view.ui
README.md        ← This file
```

---

## How to Run

> **Requirements:** Python 3.9+ and PySide6 installed.

```bash
# Install PySide6 if needed
pip install PySide6

# Run the app
python main.py
```

---

## App Usage

1. Type a temperature value in the input field (e.g. `100`).
2. Press **Convert** or hit **Enter**.
3. The result appears below the button (e.g. `100.00 °C = 212.00 °F`).
4. Non-numeric input shows a red error message.

---

## Regenerating the View

If you modify `view.ui` in Qt Designer, regenerate `view.py` with:

```bash
pyside6-uic view.ui -o view.py
```
